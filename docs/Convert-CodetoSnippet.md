# Convert-CodetoSnippet

## SYNOPSIS

Convert selected code to a snippet

## SYNTAX

```yaml
Convert-CodetoSnippet [-Text] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
